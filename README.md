# Update `config.js` :
```json
{    
    "domain": "", //replace with you domain for socket.io to make connection
    "botName": "google-PaLM2-chatbot",
    "api_key": "" //api key
}
```
for your api key.
* you can obtain it from https://aistudio.google.com/app/apikey

# To setup:
```js
npm i
```
```js
node .
```
